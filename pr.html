Pr 傻瓜式工作流（照着做）

你只做这 7 步：每步都有“成功标志”。

建议：先做一遍再回头优化
  1. 新建项目：项目路径尽量别放中文/太深路径（降低导出坑）。
  2. 导入素材：把视频/音频拖进项目面板。成功：能在面板里看到素材缩略图
  3. 新建序列：把主视频拖到时间线。成功：时间线上出现画面和音频轨道
  4. 剪切：用剃刀工具/快捷键剪掉废片。成功：视频更短，节奏更紧
  5. 加字幕：先用最简单文本字幕。成功：字幕能清晰看见，不挡脸
  6. 加音乐并对齐节奏：音乐音量别盖过人声。成功：听起来舒服，不炸耳
  7. 导出 MP4：H.264 / 1080p 起步。成功：导出文件能正常播放
导出推荐（新手最稳）：
格式：H.264 ｜预设：Match Source - High Bitrate 或 1080p ｜输出：MP4

Pr 新手必备快捷键（先记 5 个）

少点鼠标，多点效率。

  • C 剃刀（剪切）
  • V 选择工具
  • Space 播放/暂停
  • Ctrl+Z 撤销
  • Ctrl+M 导出（常用）

常见问题（卡住先看这里）

你遇到的 80% 问题都在下面。

  • 卡顿/预览不顺：先把播放分辨率调低，必要时开 Proxy（代理）。
  • 导出失败：换到英文路径、缩短路径深度、改文件名、关掉占用输出文件的软件。
  • 声音忽大忽小：先把背景音乐整体拉到更低，再微调关键帧。
  • 画面颜色不对：先别调色，先保证能正常导出播放。
